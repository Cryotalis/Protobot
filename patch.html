What's New? (Update V3 | Released on December 27th, 2021)

dd!listshards and dd!listmods

Lists all shards or mods that are equippable on a given slot on a given hero from a given difficulty.

Custom prefixes

You can now set a custom prefix or mod prefix for your server.

Blacklist and Whitelist commands

You can now blacklist or whitelist commands across the server or in certain channels. Whitelisting is slightly more advanced, see the corresponding Advanced Usage Tips and Information section for more info.

Command Aliases

Several commands (such as dd!drakenfrost, dd!image, and dd!minasc) now have aliases to make entering them easier.

dd!ask

Ask Protobot a yes or no question.

dd!choose

Have Protobot choose between 2 or more things.

dd!drunk

cpnverts texu intl drunk texu.

dd!time

Shows the current time at Chromatic Games Headquarters (Gainesville, Florida).

@Protobot

Ping Protobot to check what the current prefix and mod prefix is on your server.

For information on how to use any of these commands, please see the command list.

Improvements and Bug Fixes

  • All commands now use Discord's inline reply function to reply to the user (without @mentioning the user)
  • The formatting has been improved for the suggestion message that pops up when you accidentally used dd!shard to search for a mod or vice versa
  • dd!drakenfrost no longer lags behind the Drakenfrost Keep rotation
  • dd!drakenfrost now tells you exactly when the next rotatino at through Discord's timestamp function
  • dd!minasc now tells you exactly how many points you will have in Offense/Defense/Utility (thanks to Gigazelle for suggesting this!)
  • dd!minasc's Reset suggestion section has been improved to more properly reflect current wisdom on Ancient Power Resets
  • dd!mod will now assume that you are searching for a Servo instead of a Chip if you do not specify whether you are searching for a Servo or a Chip.
  • dd!price has seen various bug fixes and improvements:
    • It is no longer possible to search for a pet with a mod quality (e.g 10/10 Katkarot)
    • Mods that are 9/10 or lower now use proper abbreviations for numbers. (Prices that previously displayed 0.65m for example will now display 650k instead for better readability)
    • If you search for a mod and do not specify whether it is a Chip or a Servo, it will now be assumed that you wanted to search for a Servo instead of a Chip
    • If you search for a material and do not specify whether it should be Pristine, Shiny, or Plain, it will now be assumed that you wanted to search for Pristine materials rather than Plain materials
    • You can now specify a custom number of a certain material. (Ex: dd!price 50x plain motes)
