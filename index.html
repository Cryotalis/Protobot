Protobot

Discord Bot for Dungeon Defenders 2. Made by Cryo#4704.

Commands

Command Description Example Usage
dd!shard [shard name] Search for any shard in DD2. dd!shard destruction
dd!mod [mod name] Search for any mod in DD2. dd!mod defense rate
dd!price [amount] [quality/rarity] [item] Find recommended prices for items in DD2. When specifying an amount of material, the amount should be followed immediately by an x.

Advanced Usage Tips and Information

All parameters except the item are optional. Defaults for the parameters are as follows:

  • Mod Quality: 10/10
  • Mod Type: Servo
  • Material Amount: 99x (or 1x in special cases)
  • Pet Rarity: Highest Rarity (Legendary or Mythical)

Prices taken from http://bit.ly/dd2market

 dd!price 9/10 controller servo
dd!price 5x c9 amps
dd!price legendary protobud
dd!drakenfrost Shows the weapons and mods that are in rotation in Drakenfrost Keep.

Shortcut: drakenfrost ⇄ dfk 		dd!drakenfrost
dd!minasc [ascension] [floor] Calculates your minimum ascension and talent caps, and gives you a suggestion on when to reset.

Shortcut: minasc ⇄ min 		dd!minasc 6969 420
dd!listshards [hero] [slot] [difficulty] Lists all shards that are equippable on a given slot on a given hero from a given difficulty. Slot refers to relic/helmet/boots/etc; difficulty refers to Chaos tier.

Advanced Usage Tips and Information

You must specify at least one parameter. You may input "All" for heroes to list all shards that can be used by ALL heroes, and "Armor" to list shards that go on any armor slot.

Shortcut: listshards ⇄ ls

dd!listshards Monk relic
dd!listshards Mercenary armor
dd!listshards Chaos 5 weapon
dd!listmods [hero] [slot] [difficulty] Lists all mods that are equippable on a given slot on a given hero from a given difficulty. Slot here refers to armor/relic/weapon; difficulty refers to Chaos tier.

Advanced Usage Tips and Information

You must specify at least one parameter. You may input "All" for heroes to list all mods that can be used by ALL heroes. If the mod can drop from a range of chaos tiers, the mod will be included in the list as long as it is within the range.

Shortcut: listmods ⇄ lm

dd!listmods Apprentice
dd!listmods Chaos 4 Armor
dd!listmods Weapon
dd!link [resource] Summon a link to a commonly referenced resource. dd!link DPS
dd!image [name] Summon a commonly referenced image.

Shortcut: image ⇄ img 		dd!image prime
dd!faq [FAQ] Summon an answer to a frequently asked question. dd!faq patch
Command Description Example Usage
dd!ask [question] Ask Protobot a yes/no question. dd!ask is water wet?
dd!choose [option1, option2, ...] Ask Protobot to choose between 2 or more things. Options must be separated by commas.

Advanced Usage Tips and Information

Options can be separated by any character that is not a number, letter, space, or underscore. Any of these characters will work: - + = [ ] { } \ | / ' " : ; , . /

 dd!choose homework, DD2
dd!mock [text] COnverT tExt INTo mOCKinG spOngebOB tEXT. dd!mock why did they nerf rb
dd!drunk [text] Convfrt text joto erunk text. dd!drunk temple is the best map
dd!question Generates a random conversation question. dd!question
dd!time Displays the current time in Gainesville, Florida.

Advanced Usage Tips and Information

Can also be used to convert a certain date (optional) and time from one timezone to another.
Example: dd!time March 8 7pm PST to EST

Timezones taken from https://www.timeanddate.com/time/zones/

 dd!time
dd!feedback [feedback] Submit feedback or suggestions for Protobot. dd!feedback Protobot is awesome!
dd!bugreport [bug] Submit a bug report for Protobot. dd!bugreport dd!shard isn't working.
dd!ping Shows Protobot's response time. dd!ping
dd!help Displays a link to this website and a link to Protobot's support Discord server. dd!help
@Protobot Tells you the current command prefix and mod command prefix for Protobot. @Protobot
Command Description Example Usage
dd?prefix [prefix]
dd?modprefix [prefix]		 Set a custom prefix or modprefix for your server. Not providing a prefix resets the prefixes to their defaults. dd?prefix !
dd?modprefix ?
dd?disable [command] [channel(s)]
dd?enable [command] [channel(s)]		 Disable or enable a command in one or more channels. Not specifying any channels disables or enables the command in all channels.

Advanced Usage Tips and Information

You can whitelist a command in a certain channel or channels by first disabling the command in all channels and then enabling it in a certain channel or channels.

 dd?disable drakenfrost #general
dd?enable price #bot-spam
dd?disabled Shows which commands are currently disabled on the server. dd?disabled

Features

Customized Spark Profile

Granblue Fantasy themed profile card with a customizable background.

You'll get a congratulatory message when you've saved up for a spark.

Customizable Welcome Message

Customizable background and welcome message. Choose what channel the message is sent in, and choose whether to enable leave messages and ban messages.

Player Search

Search for a player by ID or player name. Displays an image containing the player's profile page, support summons, and star character.

Automatically Updated Spark Counters

Automatically updates the spark counter in your nickname as you key in your spark funds.

Accepted formats: (000/300) & 00.00%

Europa's role must be in a higher position than the roles of those whose nicknames she is trying to update in order for this to work.

This feature does not work for the server owner due to a Discord limitation.

Events

Shows current and upcoming events. Tells you how much time is left in current events and tells you when upcoming events start.
