Protobot

Discord Bot for Dungeon Defenders 2. Made by Cryo#4704.

Privacy and Data Usage Policy

I do not store or collect any user data other than what is necessary. Protobot's database is a public Google Spreadsheet. It is viewable by anyone.

The only potentially sensitive data that can be collected from the user is whatever the user sets as their command or mod command prefix. Obviously, you should not set your password or any other sensitive information as a command prefix. If you have done so by accident, running the (prefix or modprefix) command by itself without any arguments will overwrite whatever was there previously.
