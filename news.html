News and Updates (July 2022)

  • This website has received a redesign, and several new pages have been added.
  • Protobot has been verified! At the time of writing, it is used in 80 servers.
  • Support for message commands (commands that use dd! or dd?) has been discontinued. The vast majority of users have already switched to slash commands, and from now on, only slash commands will be supported. If slash commands do not work on your server, please click the "Add To Server" button. It will prompt you to enable slash commands. You don't need to kick and re-invite Protobot to enable slash commands.
  • The feedback and bugreport commands have been removed. These commands saw little use, and the few people who actually used them did not use them as they were intended to be used. Going forward, feedback and bug reports for Protobot should be sent to the Protobot Support Server instead. It's easier for me to respond to reports and feedback this way as well.
  • The custom prefix and blacklist/whitelist commands features have been removed. Since Protobot has switched to slash commands, the only prefix now is /, and slash commands can be whitelisted and blacklisted through Discord settings (Server Settings > Integrations > Protobot > Manage).
  • The old Python version of Protobot is now open source, and can be found here. It was written using discord.py, and was retired at some point in December 2021. Please note that most commands will not work if you don't connect Protobot to the public database yourself.

Got a concern, suggestion, or request? Join the Protobot Support Server.
